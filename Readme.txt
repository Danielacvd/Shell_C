Para compilar un programa en C, en la terminal:

--> Situate en Escritorio
--> pwd para comprobar donde estas
--> gcc (nombre del archivo con su extension) -o (nombre para compilar)
-->--> gcc main.c -o execute
--> Para correr el programa
--> ./execute
--> Help
--> ls
--> cd
--> mkdir
--> touch
--> exit
